# Roteiro manual offline

Use este roteiro com a internet realmente desligada.

1. Desligar Wi-Fi e cabo de rede.
2. Abrir o app.
3. Confirmar que a tela inicial carrega sem erro.
4. Abrir o caixa.
5. Criar um cliente.
6. Criar uma categoria.
7. Criar um fornecedor.
8. Criar um produto.
9. Realizar uma venda do produto criado.
10. Consultar o financeiro.
11. Consultar os relatorios.
12. Criar um backup.
13. Fechar o app.
14. Abrir o app novamente.
15. Confirmar que cliente, categoria, fornecedor, produto, caixa, venda, financeiro e relatorios permanecem gravados.
16. Confirmar que o backup criado continua listado.

Resultado esperado:

- nenhuma tela principal depende de internet;
- nenhum asset essencial quebra offline;
- os dados criados persistem apos reabertura;
- o fluxo principal segue utilizavel sem rede.
