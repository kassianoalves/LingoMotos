# Small Screen Validation

Validation target: no critical action button can leave the visible viewport.

Critical actions: Fechar/X, Cancelar, Confirmar, Salvar, Finalizar venda, Restaurar, Excluir, Resetar, Abrir caixa, Fechar caixa.
