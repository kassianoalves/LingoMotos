# Shared

Infraestrutura frontend reutilizavel entre features.

Pastas principais:

- `api`: contratos e clientes de transporte;
- `lib`: wrappers tecnicos e adaptadores;
- `query`: configuracao global do TanStack Query;
- `stores`: stores Zustand globais de UI;
- `types`: tipos compartilhados.

Codigo de negocio deve ficar em `src/features`.
